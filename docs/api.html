Problem class API¶

A pressio-demoapps problem class meets a specific API as described below.

C++ API¶

class Problem¶
type scalar_type¶

The scalar type: this is by default double.

type state_type¶

Data structure type to store the state: currently, this is an Eigen vector.

type velocity_type¶

Data structure type to store the velocity: currently, this is an Eigen vector.

type jacobian_type¶

Data structure type to store the Jacobian: currently, this is an Eigen sparse CRS matrix.

auto totalDofSampleMesh()¶

Returns the total number of degrees of freedom on the sample mesh. Note that, in general, this is not the same as the number of sample mesh cells. When you have multiple dofs/cell (for example Euler equations), then the total # of dofs on sample mesh = # dofs/cell times the # of sample mesh cells.

auto totalDofStencilMesh()¶

Returns the total number of degrees of freedom on the stencil mesh. Note that, in general, this is not the same as the number of stencil mesh cells. When you have multiple dofs/cell (for example Euler equations), then the total # of dofs on stencil mesh = # dofs/cell times the # of sample mesh cells.

state_type initialCondition()¶

Constructs and returns an instance of the initial condition for the target problem.

velocity_type createVelocity()¶

Constructs and returns an instance of the velocity.

jacobian_type createJacobian()¶

Constructs and returns an instance of the jacobian.

void velocity(const state_type &y, scalar_type time, velocity_type &v)¶

Given a state y and time time, evaluates the RHS of the system and overwrites v.

void jacobian(const state_type &y, scalar_type time, jacobian_type &J)¶

Given a state y and time time, evaluates the Jacobian of the RHS and stores it into J.

void velocityAndJacobian(const state_type &y, scalar_type time, velocity_type &v, jacobian_type &J)¶

Given a state y and time time, evaluates the RHS and its Jacobian.

Python API¶

class Problem¶
totalDofSampleMesh()¶

Returns the total number of degrees of freedom on the sample mesh. Note that, in general, this is not the same as the number of sample mesh cells. When you have multiple dofs/cell (for example Euler equations), then the total # of dofs on sample mesh = # dofs/cell times the # of sample mesh cells.

Return type

integer

totalDofStencilMesh()¶

Returns the total number of degrees of freedom on the stencil mesh. Note that, in general, this is not the same as the number of stencil mesh cells. When you have multiple dofs/cell (for example Euler equations), then the total # of dofs on stencil mesh = # dofs/cell times the # of sample mesh cells.

Return type

integer

initialCondition()¶

Constructs and returns an instance of the initial condition for the target problem.

Return type

numpy.array

createVelocity()¶

Constructs and returns an instance of the velocity.

Return type

numpy.array

createApplyJacobianResult(operand)¶

Constructs and returns an instance of the action of the Jacobian applied to operand. The result is constructed, and zeroed out before returning it.

Parameters

operand (numpy.array) – rank-1 or rank-2 operand to apply the Jacobian to.

Return type

numpy.array

velocity(y, time, v)¶

Given a state y and time time, evaluates the RHS of the system and stores it into v.

Parameters

  • y (numpy.array) – state vector

  • time (float) – evaluation time

  • v (numpy.array) – velocity to overwrite

applyJacobianResult(y, time, operand, result)¶

Given a state y and time time, this computes the action of the Jacobian applied to operand.

Parameters

  • y (numpy.array) – state vector

  • time (float) – evaluation time

  • operand (numpy.array) – rank-1 or rank-2 operand to apply the Jacobian to.

Return type

numpy.array

Note

Note how the Python interface only supports the Jacobian action. The main reason behind this is that Pybind11 does not yet allow view semantics for Python sparse matrices.
